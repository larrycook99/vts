Unconfirmed/ConfirmedTextMessage-Request
The following is a step-by-step procedure for making a standard Unconfirmed/ConfirmedTextMessage-Request.

  1. If you want to send a confirmed request, check the "Confirmed?" checkbox. If you want to send an unconfirmed request, do not check the "Confirmed?" checkbox.
  2. Type the decimal instance of the source device. VTS automatically combines the instance with a DEVICE Object_Type to form the 'Text Message Source Device'.
  3. Type the 'Message Class'. The 'Message Class' is optional for the text message, so if you leave it blank, no 'Message Class' will be included with the request. If you want to include a 'Message Class', you can type the 'Message Class' as either an Unsigned decimal or CharacterString. VTS will automatically format the proper BACnet data-type.
  4. Select the 'Message Priority' from the drop-down list.
  5. Type the text of the 'Message'.
  6. Make sure you have selected a target device.
  7. Click the "Send" button.

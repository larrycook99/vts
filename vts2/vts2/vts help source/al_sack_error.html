Selecting a Target Device

Normally you use VTS to send BACnet service requests to BACnet devices and test the responses. However, there are times when you may want to send simple positive or negative responses, for example when testing virtual terminal services. The following procedures describe how to transmit a BACnet-SimpleACK-PDU or a BACnet-Error-PDU.

BACnet-SimpleACK-PDU
The following is a step-by-step procedure for transmitting a BACnet-SimpleACK-PDU.

  1. Select the "Simple Ack PDU" radio button.
  2. Type the decimal value for the Original Invoke ID (0-255).
  3. Select the Service ACK Choice from the drop-down list.
  4. Make sure you have selected a target device.
  5. Click the "Send" button.

BACnet-Error-PDU
The following is a step-by-step procedure for transmitting a BACnet-Error-PDU.

  1. Select the "Error PDU" radio button.
  2. Type the decimal value for the Original Invoke ID (0-255).
  3. Select the Error Choice from the drop-down list.
  4. Select the Error Class from the drop-down list.
  5. Select the Error Code from the drop-down list.
  6. Make sure you have selected a target device.
  7. Click the "Send" button.

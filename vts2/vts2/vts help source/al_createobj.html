CreateObject-Request
The following is a step-by-step procedure for making a standard CreateObject-Request.

  1. Click the "CreateObject" radio button.
  2. Make sure the 'List of Initial Values' list at the bottom of the dialog is clear.
  3. Select a standard BACnet Object_Type from the drop-down list.
  4. Enter a decimal Instance for the Object_Type. VTS automatically calculates the Object_Identifier for the 'Object Specifier'. If you want the 'Object Specifier' to be an Object_Type rather than an Object_Identifier, leave the Instance blank.

  5. If you want to define a 'List of Initial Values' for the properties of the object, click the "add" button. If you do not want to define a 'List of Initial Values' for the properties of the object proceed to step 12.

In the "Edit Property Value" window:

  1. Select a standard 'Property Identifier' from the drop-down list. The property list is automatically filled in with the properties appropriate for the selected Object_Type.
  2. Type the initial value for the property into the property value edit box. Note that VTS automatically determines the type of value you can enter into the value box based on the 'Property Identifier' you select and displays the appropriate type of edit box. Click here for more information about editing individual values and here for more information about editing list values.
  3. Click the "OK" button.

The "Create/Delete Object" window is redisplayed with the initial value for the 'Property Identifier' added to the 'List of Initial Values'.

  1. Repeat steps 5-8 as needed to complete the 'List of Initial Values'.

If you need to remove values from the list:

  1. Select the 'Property Identifier' and initial value to delete in the list box.
  2. Click the "delete" button.

When your listOfInitialValues is complete:

  1. If no initial values are required for the object, the 'List of Initial Values' will be empty.
  2. Make sure you have selected a target device.
  3. Click the "Send" button.

Editing 'Property Values' that are Lists
The following is a step-by-step procedure for editing a typical 'Property Value' that is a List.

  1. Select the 'Property Identifier' from the drop-down list.
  2. Click the "add" button when the list box appears.

In the "Edit Property Value" window:

  1. Enter/select the parameters.
  2. Click the "OK" button.

  1. Repeat steps 2-4 as many times as needed to complete the list.
  2. Click the "OK" button.

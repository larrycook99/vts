AtomicReadFile-Request using Stream Access
The following is a step-by-step procedure for making a standard AtomicReadFile-Request using Stream Access.

  1. Click the "AtomicReadFile" radio button.
  2. Type the decimal file instance value. VTS automatically calculates the 'File Identifier' using the file instance and the FILE Object_Type.
  3. Click the "Stream Access" radio button.
  4. Type a decimal value for the 'File Start Position'.
  5. Type a decimal value for the 'Requested Octet Count'.
  6. Make sure you have selected a target device.
  7. Click the "Send" button.

AtomicReadFile-Request using Record Access
The following is a step-by-step procedure for making a standard AtomicReadFile-Request using Record Access.

  1. Click the "AtomicReadFile" radio button.
  2. Type the decimal file instance value. VTS automatically calculates the 'File Identifier' using the file instance and the FILE Object_Type.
  3. Click the "Record Access" radio button.
  4. Type a decimal value for the 'File Start Record'.
  5. Type a decimal value for the 'Requested Record Count'.
  6. Make sure you have selected a target device.
  7. Click the "Send" button.

ReadPropertyMultiple-Request
The following is a step-by-step procedure for making a standard ReadPropertyMultiple-Request.

  1. Make sure the "Read Multiple Properties?" box is checked.
  2. Make sure the 'List of Read Access Specifications' at the bottom of the dialog is clear.
  3. Select a standard BACnet Object_Type from the drop-down list.
  4. Enter a decimal InstanceS for the Object_Type. VTS automatically calculates the 'Object Identifier'.
  5. Select a 'Property Identifier' from the drop-down list.
  6. If you want to include an 'Property Array Index', check the "Always send array index with single property?" box. If you do check the box, an array index edit box will appear for you to fill in.
  7. Click the "add" button.

  1. Repeat steps 3-7 as needed to complete the 'List of Read Access Specifications'.

If you need to remove properties from the list:

  1. Select the item in the 'List of Read Access Specifications' to delete.
  2. Click the "delete" button.

When your 'List of Read Access Specifications' is complete:

  1. Make sure you have selected a target device.
  2. Click the "Send" button.

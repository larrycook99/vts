Unconfirmed/ConfirmedPrivateTransfer-Request
The following is a step-by-step procedure for making a standard Unconfirmed/ConfirmedPrivateTransfer-Request.

  1. If you want to send a confirmed request, check the "Confirmed?" checkbox. If you want to send an unconfirmed request, do not check the "Confirmed?" checkbox.
  2. Either type the decimal 'Vendor ID' value or select the vendor's name from the drop-down list.
  3. Type the decimal value of the 'Service Number'.
  4. If you want to include optional 'Service Parameters' with the request, type the hex OCTET STRING into the edit box. VTS automatically encloses the OCTET STRING in the open/close tags appropriate to the request. The corresponding encoding in a transmitted packet for this example is shown below for reference.

  1. Make sure you have selected a target device.
  2. Click the "Send" button.

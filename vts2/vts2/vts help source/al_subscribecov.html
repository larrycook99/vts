SubscribeCOV-Request
The following is a step-by-step procedure for making a standard SubscribeCOV-Request.

  1. Click the "Subscribe for COV Notification" radio button.
  2. Type the decimal value for the 'Subscriber Process Identifier'.
  3. Select a standard BACnet Object_Type for the 'Monitored Object Identifier' from the drop-down list.
  4. Enter a decimal Instance for the Object_Type. VTS automatically calculates the 'Monitored Object Identifier'.

If you want to subscribe or resubscribe to a COV:

  1. Click the "Subscribe/Resubscribe" radio button.
  2. If you want confirmed COV notifications, check the "Issue Confirmed Notifications?" checkbox. If you want unconfirmed COV notifications, do not check the "Issue Confirmed Notifications?" checkbox.
  3. If you want a finite 'Lifetime' for the subscription type a value in seconds into the lifetime edit box. If you want an infinite 'Lifeime', enter a 0 value or check the "permanent" checkbox.

If you want to cancel a COV subscription:

  1. Click the "Cancel Subscription" radio button.
  2. Note that an unchecked "Issue Confirmed Notifications?" checkbox and 'Lifetime'=0 are not the same as clicking the "Cancel Subscription" radio button. The former subscribes to an infinite unconfirmed subscription, while the latter cancels a subscription.

  1. Make sure you have selected a target device.
  2. Click the "Send" button.

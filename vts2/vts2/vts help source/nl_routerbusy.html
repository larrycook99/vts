BACnet Router-Busy-To-Network Message
The following is a step-by-step procedure for transmitting a BACnet Router-Busy-To-Network Message.

  1. If you want to include a single DNET, type the decimal value for the network (0-65535).
  2. Make sure you have selected a target device.
  3. Click the "OK" button.

  1. If you want to include a list of DNETs, type the comma-separated decimal values for the networks (0-65535).
  2. Make sure you have selected a target device.
  3. Click the "OK" button.

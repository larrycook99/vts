BACnet Establish-Connection-To-Network Message
The following is a step-by-step procedure for transmitting a BACnet Establish-Connection-To-Network Message.

  1. Type the decimal value for the DNET to which you want to establish a connection (0-65535).
  2. If you want a connection that is broken after a period of NPDU inactivity on the connection, make sure the "Permanent" checkbox is not checked and type a non-zero decimal value in seconds for the Termination Time Value.

    If you want a permanent connection, make sure the "Permanent" checkbox is checked or you type a value of zero for the Termination Time Value.

  3. Make sure you have selected a target device.
  4. Click the "OK" button.

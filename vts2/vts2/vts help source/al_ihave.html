I-Have-Request
The following is a step-by-step procedure for making a standard I-Have-Request.

  1. Click the "I-Have" radio button.
  2. Type a decimal Device Instance for the 'Device Identifier'. VTS automatically adds the DEVICE Object_Type to the Device Instance to calculate the 'Device Identifier'.
  3. Select a standard BACnet Object_Type from the drop-down list.
  4. Type a decimal Instance for the Object_Type. VTS automatically calculates the 'Object Identifier'.
  5. Type the 'Object Name' into the object name edit box.
  6. Make sure you have selected a target device.
  7. Click the "Send" button.

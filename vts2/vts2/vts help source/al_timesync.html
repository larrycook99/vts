TimeSynchronization-Request
The following is a step-by-step procedure for making a standard TimeSynchronization-Request.

VTS automatically fills in the current time and date when you select the Time Synchronization menu.

  1. If you want to change the date portion of the BACnetDateTime for the 'Time' parameter from the current date, use the drop-down lists change the day, month and weekday. Use the edit box to change the year.
  2. If you want to change the time portion of the BACnetDateTime for the 'Time' parameter from the current time, type the time in hr:min:sec.hundredths in 24-hour format.
  3. Make sure you have selected a target device.
  4. Click the "Send" button.

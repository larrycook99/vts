ReadProperty-Request
The following is a step-by-step procedure for making a standard ReadReadProperty-Request.

  1. Select a standard BACnet Object_Type from the drop-down list.
  2. Enter a decimal Instance for the Object_Type. VTS automatically calculates the 'Object Identifier'.
  3. Select a standard 'Property Identifier' from the drop-down list. The property list is automatically filled in with the properties appropriate for the selected Object_Type.
  4. If you want to include an 'Property Array Index', check the "Always send array index with single property?" box. If you do check the box, an "array index" edit box will appear for you to fill in.
  5. Make sure you have selected a target device.
  6. Click the "Send" button.

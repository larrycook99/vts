VTS Configuration

Setting up VTS
Before you can use VTS you must setup the following files:

  • VTS.INI is the main VTS initialization file
  • VTSAlias.INI is a BACnet target device alias file
  • VTSRoute.INI is a BACnet router alias file
  • VTSBind.INI is a BACnetRecipient alias file
  • NALSERV32.INI or preferably the NALSERV32 Configuration Menu
  • MSTP.INI if you are using BACnet MS/TP
  • PTP.INI if you are using BACnet PTP

VTS.INI
VTS.INI is the primary initialization file that defines the parameters used by VTS. A default VTS.INI is saved in the \Windows or \WINNT folder when VTS is installed. The default VTS.INI normally does not require modification, however in case you need to make changes, the parameters of VTS.INI are explained below:

  1. [General] is the main section header for the file.
  2. Version defines the version number of VTS (normally set to 2.00)
  3. DLLs defines an absolute path to the VTS support DLLS including NALSERV32.DLL, BACPRIM32.DLL, etc.
  4. [VendorID] is the vendor section header for the file.
  5. MaxID identifies the maximum vendor ID number in the vendor list which follows.
  6. vendorid=VendorName defines the vendors by their Vendor_Identifier and Vendor_Name. Note that because of screen size restrictions, all of the current vendors are not shown in the list above .

Target Device and Router Alias Files
Normally these files are created using the Targer Device dialog.

  1. Type the hex MAC address for the target device or router.
  2. Type the alias you wish to use for the device or router.
  3. Click the appropriate "new alias" button.

The new alias is added to the VTSAlias.INI file or the VTSRoute.INI file. Both files are plain-text files and are editable using any plain-text editor such as Notpad.

BACnetRecipient Binding Alias Files
Normally these files are created using the BACnetRecipient dialog.

  1. Type the decimal value for the network-number of the BACnetRecipient's BACnetAddress.
  2. Type the hex value for the mac-address of the BACnetRecipient's BACnetAddress.
  3. Type the alias you wish to use for the recipient.
  4. Click the appropriate "new alias" button.

The new alias is added to the VTSBind.INI file, which is a plain-text file and can be edited using any plain-text editor such as Notepad.

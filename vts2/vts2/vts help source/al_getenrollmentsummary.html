GetEnrollmentSummary-Request
The following is a step-by-step procedure for making a standard GetEnrollmentSummary-Request.

  1. Select the 'Acknowledgement Filter' from the radio buttons.

If you want to include the optional 'Enrollment Filter' in the request, and if the BACnetRecipient in the BACnetRecipientProcess is a device then:

  1. Check the "Enrollment Filter" checkbox.
  2. Select the "Device Instance" radio button.
  3. Type the decimal value of the Device Instance for the BACnetRecipient.
  4. Type the decimal value for the Process Identifier of the BACnetRecipientProcess.

If you want to include the optional 'Enrollment Filter' in the request, and if the BACnetRecipient in the BACnetRecipientProcess is a network/mac address then:

  1. Check the "Enrollment Filter" checkbox.
  2. Select the "Network/MAC Address" radio button.
  3. Type the decimal value of the network-number and the hex value of the mac-address for the BACnetRecipient.
  4. Type the decimal value for the Process Identifier of the BACnetRecipientProcess.

If you want to include the optional 'Event State Filter' in the request then:

  1. Check the "Use Event State Filter?" checkbox.
  2. Select the 'Event State Filter' from the drop-down list.

If you want to include the optional 'Event Type Filter' in the request then:

  1. Check the "Use Event Type Filter?" checkbox.
  2. Select the 'Event Type Filter' from the drop-down list.

If you want to include the optional 'Priority Filter' in the request then:

  1. Check the "Priority Filter" checkbox.
  2. Type the decimal value of the MinPriority (0-255).
  3. Type the decimal value of the MaxPriority (0-255).

If you want to include the optional 'Notification Class Filter' in the request then:

  1. Check the "Notification Class Filter" checkbox.
  2. Type the decimal value of the 'Notification Class Filter'.

  1. Make sure you have selected a target device.
  2. Click the "Send" button.

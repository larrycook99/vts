Who-Has-Request
The following is a step-by-step procedure for making a standard Who-Has-Request.

  1. Click the "Who-Has" radio button.
  2. If you want to specify the optional 'Device Instance Range Low Limit' and 'Device Instance Range High Limit' for a qualified request, make sure the "Device Instance Range?" box is checked and enter decimal Device Instance values for the Low Limit and High Limit. Otherwise, leave the box unchecked.

If you want to specify the object by 'Object Name':

  1. Make sure the "Name" box is checked.
  2. Type the 'Object Name' into the object name edit box.
  3. Make sure you have selected a target device.
  4. Click the "Send" button.

If you want to specify the object by 'Object Identifier':

  1. Make sure the "ObjID" box is checked.
  2. Select a standard BACnet Object_Type from the drop-down list.
  3. Enter a decimal Instance for the Object_Type. VTS automatically calculates the 'Object Identifier'.
  4. Make sure you have selected a target device.
  5. Click the "Send" button.

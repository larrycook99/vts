ReadPropertyConditional-Request
The following is a step-by-step procedure for making a standard ReadPropertyConditional-Request.

  1. Click the "Edit" button.

In the "Edit Property Value" window:

  1. Select a standard 'Property Identifier' from the drop-down list. All the standard BACnet properties are included in this list as well as selections for OPTIONAL and REQUIRED.
  2. Select the 'Relation Specifier' from the drop-down list.
  3. Type the 'Comparison Value' into value edit box. Note that VTS automatically determines the type of constant value you can enter into the edit box based on the 'Property Identifier' you select and displays the appropriate type of edit box.
  4. Click the "OK" button.

The "property identifier (rel) value" section, is filled in with you selections from steps 2-5. If you want to select multiple criteria for the 'List of Selection Criteria', go to step 6. If you do not want to select multiple criteria for the 'List of Selection Criteria' go to step 13.

  1. Make sure the "Multiple Criteria?" box is checked.
  2. Make sure the 'List of Selection Criteria' at the bottom of the dialog is clear.
  3. Make a 'Selection Logic' choice from the drop-down list at the top of the dialog.
  4. Click the "add" button.

  1. Repeat steps 1-5 and 8 as needed to complete the 'List of Selection Criteria'.

If you need to remove conditions from the list:

  1. Select the condition in 'List of Selection Criteria' to delete.
  2. Click the "delete" button.

If you want to include an optional 'List of Property References', go to step 13. If you do not want to include a 'List of Property References', i.e. you want ALL properties of an object that match the selection criteria, go to step 23.

  1. Make sure the "Include Reference List?" box is checked.
  2. Click the "Refs" button.

If you want a single BACnetPropertyReference, go to step 15. If you want multiple BACnetPropertyReferences, go to step 17.

In the "Edit Property References" window:

  1. Select a standard 'Property Identifier' from the drop-down list.
  2. Click the "OK" button.
Go to step 23.

In the "Edit Property References" window:

  1. Make sure the "Multiple References?" box is checked.
  2. Make sure the 'List of Property References' at the bottom of the dialog is clear.
  3. Select a standard 'Property Identifier' from the drop-down list.
  4. Click the "add" button.

  1. Repeat steps 19-20 until your 'List of Property References' is complete.

When you've completed the 'List of Property References':

  1. Click the "OK" button.

When you've completed the 'List of Selection Criteria' and 'List of Property References':

  1. Make sure you have selected a target device.
  2. Click the "Send" button.

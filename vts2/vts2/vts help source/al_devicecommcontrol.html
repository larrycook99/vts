DeviceCommunicationControl-Request
The following is a step-by-step procedure for making a standard DeviceCommunicationControl-Request.

  1. Select Disable Device or Enable Device option from the drop-down 'Enable/Disable' list.
  2. If the request needs to have a 'Time Duration', which is optional, type the time of the duration in minutes. If the request does not need a 'Time Duration', leave the duration blank.
  3. If the request needs a 'Password', which is optional, type the 'Password'. If the request does not need a 'Password', leave the 'Password' blank.
  4. Make sure you have selected a target device.
  5. Click the "Send" button.

I-Am-Request
The following is a step-by-step procedure for making a standard I-Am-Request.

This dialog allows you to send any I-Am-Request with any combination of parameters. This is different than the I-Am-Requests that are sent by NALSERV32, which are strictly intended to identify the VTS workstation as a BACnet device on a BACnet network.

  1. Click the "I-Am" radio button.
  2. Type the decimal Device Instance value for the request. VTS automatically adds the DEVICE Object_Type to the Device Instance to form the 'I-Am Device Identifier' for the request.
  3. Type the decimal 'Max APDU Length Accepted' value for the request.
  4. Select the 'Segmentation Supported' from the drop-down list.
  5. Either type the decimal 'Vendor Identifier' value or select the vendor's name from the drop-down list.
  6. Make sure you have selected a target device.
  7. Click the "Send" button.

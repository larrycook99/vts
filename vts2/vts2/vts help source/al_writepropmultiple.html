WritePropertyMultiple-Request
The following is a step-by-step procedure for making a standard WritePropertyMultiple-Request.

  1. Select Write Property from the services drop-down list.
  2. Make sure the "Write Multiple Properties?" box is checked.
  3. Make sure the 'List of Write Access Specifications' at the bottom of the dialog is clear.
  4. Select a standard BACnet Object_Type from the drop-down list.
  5. Enter a decimal Instance for the Object_Type. VTS automatically calculates the 'Object Identifier'.
  6. Click the "edit" button.

In the "Edit Property Value" window:

  1. Select a standard 'Property Identifier' from the drop-down list. The property list is automatically filled in with the properties appropriate for the selected Object_Type.
  2. Type the 'Property Value' to write into the property value edit box. Note that VTS automatically determines the type of 'Property Value' and possible 'Property Array Index' you can enter based on the 'Property Identifier' and displays the appropriate type of edit box. Click here for more information about editing individual values and here for more information about editing list values.
  3. Select the optional 'Priority' and check the "Relinquish?" box if appropriate.
  4. Click the "OK" button.

The "Write Property Multiple" window is redisplayed with the 'Property Identifier' and the 'Property Value' to write to it is displayed.

  1. Click the "add" button.

  1. Repeat steps 4-11 as needed to complete the 'List of Write Access Specifications'.

If you need to remove values from the list:

  1. Select the 'Write Access Specification' item in the 'List of Write Access Specifications' to delete.
  2. Click the "delete" button.

When your 'List of Write Access Specifications' is complete:

  1. Make sure you have selected a target device.
  2. Click the "Send" button.

Editing 'Property Values' that are Lists
The following is a step-by-step procedure for editing a typical 'Property Value' that is a List.

  1. Select the 'Property Identifier' from the drop-down list.
  2. Click the "add" button when the list box appears.

In the "Edit Property Value" window:

  1. Enter/select the parameters.
  2. Click the "OK" button.

  1. Repeat steps 2-4 as many times as needed to complete the list.
  2. Click the "OK" button.

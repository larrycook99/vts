AcknowledgeAlarm-Request
The following is a step-by-step procedure for making a standard AcknowledgeAlarm-Request.

  1. Type the decimal value for the 'Acknowledging Process Identifier'.
  2. Type the 'Acknowledgement Source' as a CharacterString.
  3. Select the 'Event State Acknowledged' from the drop-down list of event states.
  4. Select a standard BACnet Object_Type for the 'Event Object Identifier' from the drop-down list.
  5. Enter a decimal Instance for the Object_Type. VTS automatically calculates the 'Event Object Identifier' by combining the Object_Type and Instance.

The 'Time Stamp' and the 'Time Of Acknowledgement' are both BACnetTimeStamp data types. The BACnetTimeStamp can be one of three choices:

  1. time
  2. sequenceNumber
  3. dateTime
The following examples show both the 'Time Stamp' and 'Time Of Acknowledgement' as the same choice of data type for the BACnetTimeStamp. This is for convenience of explanation only. In fact, the 'Time Stamp' and 'Time Of Acknowledgement' may be different data type choices.

If the 'Time Stamp' or 'Time Of Acknowledgement' is a dateTime choice for the BACnetTimeStamp:

  1. Select the "date/time" radio button.
  2. Select the day, month and weekday from the drop-down lists. Type the year and time. As you type, invalid entries are shown in red, while valid entries are shown in green.

If the 'Time Stamp' or 'Time Of Acknowledgement' is a time choice for the BACnetTimeStamp:

  1. Select the "time" radio button.
  2. Type the time. As you type, invalid entries are shown in red, while valid entries are shown in green.

If the 'Time Stamp' or 'Time Of Acknowledgement' is a sequenceNumber choice for the BACnetTimeStamp:

  1. Select the "sequence" radio button.
  2. Type the decimal value for the sequenceNumber. As you type, invalid entries are shown in red, while valid entries are shown in green.

  1. Make sure you have selected a target device.
  2. Click the "Send" button.

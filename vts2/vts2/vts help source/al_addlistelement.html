AddListElement-Request
The following is a step-by-step procedure for making a standard AddListElement-Request.

  1. Select a standard BACnet Object_Type from the drop-down list.
  2. Enter a decimal Instance for the Object_Type. VTS automatically calculates the 'Object Identifier'.
  3. Select a 'Property Identifier' from the drop-down list. Note that only properties that are BACnet list types appear in the scrolling list.
  4. Make sure the 'List of Elements' at the bottom of the dialog is clear.
  5. Click the "add" button.

In the "Edit Property Value" window:

  1. Click the "Add" button.

The type of window displayed at this step depends on the type of list. The "Device Address Binding" window shown here is representative of the type of window used to enter the parameters for list elements. In the resulting window:

  1. Enter the parameters for the list element.
  2. Click the "OK" button.

In the "Edit Property Value" window:

  1. Repeat steps 6-8 as needed to complete the 'List of Elements'.

In the "Edit Property Value" window:

  1. Click the "OK" button.

  1. Make sure you have selected a target device.
  2. Click the "Send" button.

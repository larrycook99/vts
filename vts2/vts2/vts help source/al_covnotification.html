Unconfirmed/ConfirmedCOVNotification-Request
The following is a step-by-step procedure for making a standard Unconfirmed/ConfirmedCOVNotification-Request.

  1. Click the "Issue COV Notification" radio button.
  2. If you want to send a confirmed request, check the "Confirmed?" checkbox. If you want to send an unconfirmed request, do not check the "Confirmed?" checkbox.
  3. Type the decimal value for the 'Subscriber Process Identifier'.
  4. Select a standard BACnet Object_Type for the 'Monitored Object Identifier' from the drop-down list.
  5. Enter a decimal Instance for the Object_Type. VTS automatically calculates the 'Monitored Object Identifier'.
  6. Type the decimal value of the Device Instance for the 'Initiatiating Device Identifier'.
  7. Type the 'Time Remaining' in seconds.
  8. Click the "Add" button.

In the "Edit Property Value" window:

  1. Select a standard 'Property Identifier' from the drop-down list. The property list has been automatically filled in with the properties appropriate for the selected Object_Type.
  2. Type the value to send for the property into the 'Property Value' edit box. Note that VTS automatically determines the type of value you can enter into the value box based on the 'Property Identifier' you select and displays the appropriate type of edit box. Click here for more information about editing individual values and here for more information about editing list values.
  3. Click the "OK" button.

  1. Repeat steps 8-11 as needed to complete the 'List of Values'.

If you need to remove or change properties in the list:

  1. Select the Property Identifier=value to delete in the list box.
  2. Click the "Delete" button or "Edit" button.

When your 'List of Values' is complete:

  1. Make sure you have selected a target device.
  2. Click the "Send" button.

Editing 'Property Values' that are Lists
The following is a step-by-step procedure for editing a typical 'Property Value' that is a list.

  1. Select the 'Property Identifier' from the drop-down list.
  2. Click the "Add" button when the list box appears.

In the "Edit Property Value" window:

  1. Enter/select the parameters.
  2. Click the "OK" button.

  1. Repeat steps 2-4 as many times as needed to complete the list.
  2. Click the "OK" button.

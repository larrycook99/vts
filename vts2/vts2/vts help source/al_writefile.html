AtomicWriteFile-Request using Stream Access
The following is a step-by-step procedure for making a standard AtomicWriteFile-Request using Stream Access.

  1. Click the "AtomicWriteFile" radio button.
  2. Type the decimal file instance value. VTS automatically calculates the 'File Identifier' using the file instance and the FILE Object_Type.
  3. Click the "Stream Access" radio button.
  4. If you do not want to append to the file, make sure the "Append" box is unchecked. If you do want to append to the file, make sure the "Append" box is checked.

  5. If you are not appending to the file, type a decimal value for the 'File Start Position'. If you are appending to the file, "File Start Position=EOF" will be shown in the File Start Position box.

  1. Select either hex or text data by clicking the "Hex" or "Text" button.
  2. Type the 'File Data' in the appropriate format, where hex results in an OCTET STRING and text results in a CharacterString. The Octet Count is automatically updated as you type.
  3. Make sure you have selected a target device.
  4. Click the "Send" button.

AtomicWriteFile-Request using Record Access
The following is a step-by-step procedure for making a standard AtomicWriteFile-Request using Record Access.

  1. Click the "AtomicWriteFile" radio button.
  2. Type the decimal file instance value. VTS automatically calculates the 'File Identifier' using the file instance and the FILE Object_Type.
  3. Click the "Record Access" radio button.
  4. If you do not want to append to the file, make sure the "Append" box is unchecked. If you do want to append to the file, make sure the "Append" box is checked.

  5. If you are not appending to the file, type a decimal value for the 'File Start Record'. If you are appending to the file, "File File Record=EOF" will be shown in the File Start Record box.

  1. Select either hex or text data by clicking the "Hex" or "Text" button.
  2. Type the data for a record in the appropriate format, where hex results in an OCTET STRING and text results in a CharacterString.
  3. Click the "Add" button. The data for the record is added to the 'File Record Data' box and the Record Count is automatically updated.

  1. Repeat Steps 7-8 until the all the data records you want to write have been added to the 'File Record Data'.

If you need to remove data from the 'File Record Data':

  1. Select the record to delete in the 'File Record Data' box.
  2. Click the "Delete" button.

When your list of file records to write is complete:

  1. Make sure you have selected a target device.
  2. Click the "Send" button.

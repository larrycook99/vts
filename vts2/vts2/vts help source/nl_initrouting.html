BACnet Initialize-Routing-Table Message
The following is a step-by-step procedure for transmitting a BACnet Initialize-Routing-Table Message.

  1. Make sure the "Query" checkbox is not checked.
  2. Type the decimal value of the Port ID (0-255). Remember, if you type 0 for the Port ID, then you are requesting that the receiving device should clear its routing table for the requested Connected DNET.
  3. Type the decimal value for the Connected DNET (0-65535).
  4. Type the Port Info as a character-string. VTS automatically calculates the Port Info Length.
  5. Click the "Add" button. VTS adds the routing information to the routing table and automatically calculates the Number of Ports.

  1. Repeat steps 2-5 as needed to complete the routing table information.

If you need to remove entries in the routing table list:

  1. Select the line to delete in the list box.
  2. Click the "Delete" button.

When your routing table is complete:

  1. Make sure you have selected a target device.
  2. Click the "Send" button.

Requesting Routing Table Information
If you want to request routing table information from a device instead of initializing, clearing or adding to a routing table:

  1. Make sure the "Query" checkbox is checked. Note, VTS automatically sets the Number of Ports to a value of zero when you check the "Query" checkbox.
  2. Make sure you have selected a target device.
  3. Click the "Send" button.
